Privacy posture

What this app does. And what it deliberately doesn't.

What it does

  • Read the local file you drag in. Sequential bytes, no copies, no thumbnails.
  • Strip the audio track from video locally via the bundled ffmpeg sidecar.
  • Upload that audio over a single TLS 1.3 connection to api.assemblyai.com.
  • Poll the transcript endpoint until the job completes.
  • Show the result in this window. Hold it in memory until you close the app.

What it never does

  • Phone home for licence checks. Verification is offline Ed25519.
  • Ship usage telemetry. No analytics SDK, no Sentry, no Mixpanel.
  • Auto-upload to cloud storage. Nothing gets backed up unless you do it.
  • Index file metadata for "search" features. Your library is not our product.
  • Train models on your audio. AssemblyAI's contract forbids it.

The network connection list is short on purpose. If you run Little Snitch or Lulu, you will see api.assemblyai.com:443 when a transcription starts. That's the whole network surface.
